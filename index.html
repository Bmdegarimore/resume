Aviation Flashcard Website

The Aviation department at Green River College needed an online learning resource for international students who were having trouble with aviation terminology and pronounciation.

My team built a website and a mySQL database for the project incorporating practices we learned in our Web Development class. The project spanned up to 8 weeks and in that time we built a Flash card section, memory matching game, quiz section, an administration site which the admin can add new sections, cards, and audio. We also built PHP scripts to handle user authentication and administration sessions from client side to the database.
